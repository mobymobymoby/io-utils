# parse number with bar
"1-2,   1-4, 23-45 \n" -> {100002, 100004, 230045}
"1-2, 3-4, 5-" -> FALSE
- 숫자 사이에 `-` 로 구분되는 경우
- 애초에 포맷깨졌으면 그냥 전체를 무시하고
- 모두 N-M 포맷에 맞으면 N * 100000 + M 
    - N - M 같은 포맷도 허용해주자..
